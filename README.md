ApexPlanet_DataAnalytics_# Task 3 - Deep-Dive Analysis & Interactive Dashboarding

## Internship
ApexPlanet Software Pvt. Ltd.
Data Analytics Internship (60 Days)

## Objective
Perform a deep-dive analysis on the dataset and build an interactive Power BI dashboard to visualize business insights using key performance indicators (KPIs).

## Dataset
ApexPlanet_DataAnalytics_Dataset.xlsx

## Tools Used
- Python
- Pandas
- Matplotlib
- Power BI
- Microsoft Excel

## KPIs
- Total Sales
- Total Orders
- Total Customers
- Average Sales

## Analysis Performed
- Sales by Category
- Top 10 Products
- Monthly Sales Trend
- Sales by Gender
- Sales by City

## Power BI Dashboard
The dashboard includes:
- KPI Cards
- Bar Chart (Sales by Category)
- Column Chart (Top Products)
- Line Chart (Monthly Sales Trend)
- Pie Chart (Sales by Gender)
- Bar Chart (Sales by City)
- Slicers for Date, Category, and Gender

## Key Insights
- Identified top-selling products.
- Compared sales across different categories.
- Analyzed monthly sales performance.
- Compared sales based on gender.
- Identified cities with the highest sales.
